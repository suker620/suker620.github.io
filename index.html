Hexo安装教程

#安装node.js
www.cnblogs.com/liuqiyun/p/8133904.html
node.js安装教程

#安装git
https://blog.csdn.net/weixin_45783947/article/details/105580604
git安装教程

#查看相关软件
node -v
npm -v
git –version

生成ssh: ssh-keygen -t rsa -C “ningwei_1122@163.com“

/c/Users/kiro/.ssh/id_rsa
用notepad++打开：
C:\Users\kiro.ssh\id_rsa.pub
复制全部内容
github添加ssh_key
https://github.com/settings/keys

测试绑定是否成功：
ssh -T git@github.com

安装hexo
npm install -g hexo
hexo init #初始化hexo
hexo g #生成
hexo s #启动服务
http://localhost:4000 #测试

修改：_config.yml 文件
deploy:
type: ‘git’
repository: ‘https://github.com/suker620/suker620.github.io.git'
branch: ‘main’

安装工具：（进入blog文件夹）
npm install hexo-deployer-git –save

Run
git config –global user.email “you@example.com“
git config –global user.name “Your Name”

hexo d #上传至服务器
