docker部署apache+php

发表于 2021-12-28

#1、拉取镜像
docker pull php:7.0-apache
#2、创建src目录，存放web目录
mkdir /data/src
#3、创建测试文件
vim /data/src/info.php

#4、创建容器
docker run -itd -v /data/src:/var/www/html -p 80:80 php:7.0-apache
#5、测试访问
http://192.168.3.5/info.php

发表于 2021-12-28

www.cnblogs.com/liuqiyun/p/8133904.html
node.js安装教程

https://blog.csdn.net/weixin_45783947/article/details/105580604
git安装教程

node -v
npm -v
git –version

生成ssh: ssh-keygen -t rsa -C “ningwei_1122@163.com“

/c/Users/kiro/.ssh/id_rsa
用notepad++打开：
C:\Users\kiro.ssh\id_rsa.pub
复制全部内容
github添加ssh_key
https://github.com/settings/keys

测试绑定是否成功：
ssh -T git@github.com

npm install -g hexo
hexo init #初始化hexo
hexo g #生成
hexo s #启动服务
http://localhost:4000 #测试

修改：_config.yml 文件
deploy:
type: ‘git’

branch: ‘main’

npm install hexo-deployer-git –save

git config –global user.email “you@example.com“
git config –global user.name “Your Name”

hexo d #上传至服务器
